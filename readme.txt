Este programa es la versión alpha del Sistema de Gestión de Reservas del IES Celia Viñas.

Su propósito es permitir que el profesorado pueda reservar equipos e instalaciones (tales como carritos de portátiles o aulas de informática) sin necesidad de usar papel y desde cualquier dispositivo con conexión a internet.

El código original es un proyecto de clase de 2º DAW y fue programado por Pablo Salvador Martínez (pablosmr7) durante el curso 2021/2022. El repositorio original es: https://github.com/pablosmr7/reservas
